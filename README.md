# mp_dr_connect4

A CoD4 deathrun endroom of the classic board game, Connect4

### Features

- 3 Different board sizes (small, medium, large)
- Each board is a prefab, no need to mess with anything
- Animated discs
- Optional timer, allows for timed games that don't take up the whole round
- Takes care of all spawning/teleporting
- Includes basic hud elements for telling players whos turn it is and the time remaining
